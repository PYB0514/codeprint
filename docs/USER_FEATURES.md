# Codeprint — 사용자 관점 기능 요약

> 개발자용 API 인벤토리 → [`FEATURES.md`](FEATURES.md) · 제품 개요 → [`PROJECT.md`](PROJECT.md)
>
> "지금 이 서비스에서 사용자가 뭘 할 수 있는가"를 역할별로 정리. 구현 방식이 아니라 화면에서 실제로 클릭·경험하는 것 기준.

---

## 로그인 없이 할 수 있는 것

- 랜딩 페이지에서 아무 공개 GitHub 레포 URL이나 입력해 즉시 분석 시작(로그인은 결과 저장 시점에만 요구)
- "오늘의 공개레포" — 매일 자동 선정된 유명 오픈소스 5개의 그래프를 바로 열람
- 공유 링크(`/share/{projectId}`)로 공개된 프로젝트 그래프를 로그인 없이 열람 — 레이아웃 전환, 도메인/레이어 범례, 노드 검색, **흐름 재생**, 경고 패널까지 소유자 화면과 동등하게 이용 가능(저장/수정만 제외)
- 커뮤니티 피드 열람, 게시글에 첨부된 그래프 스냅샷 열람(경고 확인·노드 코멘트 읽기 포함)
- 다른 사용자의 공개 프로필(공개 프로젝트·공개 글·팔로워 수) 열람
- 변경 이력(`/changelog`), 서비스 소개(`/how-it-works`), 발전 과정(`/evolution`), 자체 도그푸딩 사례(`/dogfooding`) 열람

## 로그인 후 — 내 프로젝트

- GitHub 레포를 연결해 프로젝트 생성 → 자동으로 파일 구조·함수 호출 흐름·DB 연결이 인터랙티브 그래프로 분석됨(평균 10~60초)
- 브랜치 선택 분석, 재분석(최신 커밋 반영), 최신 여부(freshness) 확인
- 마이페이지(`/mypage`)에서 내 프로젝트와 내가 쓴 글을 한곳에서 관리, 공개/비공개 전환, 삭제
- 그래프 화면에서:
  - 계층형/도메인 뷰 전환, 이름/주석 라벨 전환, 노드 검색
  - 함수 클릭 → 호출자/피호출자 확인, **흐름 재생**(API→함수→DB까지 단계별 재생, 분기 선택)
  - 경고 패널에서 15종 구조 경고 확인, 경고별 숨기기/패턴 예외 처리
  - "아키텍처 의도" — 원하는 모듈 경계·금지 의존 규칙을 직접 선언해 위반 시 경고 받기(선택 기능, opt-in)
  - 노드에 코멘트 남기기, 커스텀 이름/메모 지정, 배경색 지정
  - AI 설명/코드 생성(본인 API 키 등록 시) — Claude/ChatGPT/Gemini 중 선택
  - 그래프 버전 기록 열람, 특정 버전 고정(핀, 최대 5개), 두 버전 diff 비교
  - PNG로 그래프 내보내기, 경고를 마크다운으로 내보내기
  - 초대 코드로 실시간 협업(커서 공유·팀 채팅) 시작

## 커뮤니티

- 글 작성(마크다운, 피드백 유형 선택, 파일 첨부, 프로젝트 그래프 스냅샷 첨부 — 슬롯별 레이아웃 프리셋 지정 가능)
- 전체/팔로잉/갤러리(그래프 첨부 글만) 탭, 검색, 좋아요·북마크·댓글
- 다른 사용자 팔로우, 프로필 방문, DM 보내기
- 내 북마크 모아보기(`/bookmarks`)

## 결제·업그레이드

- 후원(1천~3만원 프리셋 또는 직접 입력, 토스페이먼츠)
- 팀 생성 시 좌석 결제(좌석당 4,900원/월), 좌석 추가 시 차액만 결제

## 팀 관리

- 팀 생성·이름 변경·삭제(소유자)
- 멤버 초대/제거, 역할 관리
- 프로젝트별 좌석 배분

## 계정 설정

- 프로필 이미지·그래프 배경 이미지 업로드(5MB 제한)
- AI 제공자별 API 키 등록/삭제(BYO-key, 우리 서버 비용 0)
- 알림 설정(팀챗/DM 알림 on/off), 계정 삭제

## 관리자 전용

- 전체 통계, 유저 목록·정지/재개, 플랜 변경(감사 로그 남음)
- 공지사항 게시, 일일 운영 다이제스트(신규가입·분석성공률·결제액 등), 피드백 트리아지
